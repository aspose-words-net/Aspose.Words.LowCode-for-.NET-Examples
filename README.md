## Overview

This repository contains verified C# examples for the **Aspose.Words for .NET LowCode API** — a set of
high-level classes in the `Aspose.Words.LowCode` namespace that perform common
document-processing operations in a single method call.

Each example is generated by the [lowcode-example-generator](https://github.com/babar-raza/lowcode-example-generator)
pipeline, validated through dotnet build + runtime execution + output verification, and published
here as a repeatable reference.


**Controlled pilot scope:** This repository covers Converter, Watermarker, Splitter, Replacer operations.
Broader generation requires resolving open follow-up taskcards first.

## How to Run

Clone the repository and run any example:

```bash
git clone https://github.com/aspose-words-net/Aspose.Words.LowCode-for-.NET-Examples.git
cd Aspose.Words.LowCode-for-.NET-Examples

# Restore all packages
dotnet restore

# Run a specific example (replace <example-name> with any name from the table above)
dotnet run --project examples/words/lowcode/<example-name>
```

Each example is a self-contained .NET project. Running it produces an output file in the project
directory (e.g., `output.pdf`, `output.xlsx`, `output.html`).
